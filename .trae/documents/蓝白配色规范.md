# 蓝白配色规范（全站主题）

## 目标
- 统一全站为“蓝 + 白（高对比的蓝白体系）”视觉基调：白色背景 + 深色文字 + 蓝色品牌高光。
- 所有页面背景与卡片表面仅使用下述 token，不再使用硬编码色值。

## 设计 Token（对应 CSS 变量）
- 背景（Base）
  - `--c-bg`: `rgb(248 250 252)` / `#F8FAFC`
  - `--c-surface`: `rgb(255 255 255)` / `#FFFFFF`
  - `--c-surface-2`: `rgb(241 245 249)` / `#F1F5F9`
- 文字（Text）
  - `--c-ink`: `rgb(15 23 42)` / `#0F172A`
  - `--c-muted`: `rgb(71 85 105)` / `#475569`
  - `--c-warm`: `rgb(30 64 175)` / `#1E40AF`
- 品牌（Brand）
  - `--c-brand`: `rgb(30 64 175)` / `#1E40AF`
  - `--c-brand-deep`: `rgb(30 58 138)` / `#1E3A8A`
  - `--c-brand-light`: `rgb(96 165 250)` / `#60A5FA`
- 科技强调（Accent）
  - `--c-tech`: `rgb(14 165 233)` / `#0EA5E9`
  - `--c-tech-deep`: `rgb(3 105 161)` / `#0369A1`
  - `--c-tech-light`: `rgb(125 211 252)` / `#7DD3FC`

## 使用规范（Tailwind）
- 页面背景
  - 主背景：`bg-cyber-dark`
  - 次级背景：`bg-cyber-darker`
  - 表面层：`bg-cyber-surface`
- 文本
  - 主文字：`text-ink`
  - 次要文字：`text-muted/70 ~ text-muted/85`
- 品牌色
  - 主色：`text-brand-primary` / `bg-brand-primary`
  - 强调：`text-brand-accent` / `bg-brand-accent`

## 渐变建议（可选）
- 品牌渐变：`from-brand-primary via-brand-primaryLight to-brand-accent`
- 科技渐变：`from-tech via-tech-light to-brand-accent`

## 约束
- 禁止使用不受控的蓝紫大面积渐变作为背景底色；渐变只用于文本高光、描边、分割线与少量装饰光晕。
